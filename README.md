# 🚗 Car Price Prediction using Machine Learning

This project is a Machine Learning model that predicts the price of used cars based on different features such as mileage, fuel type, transmission, engine size, and more.

---

## 📊 Project Overview

The goal of this project is to build a regression model that can accurately estimate car prices using historical data.

---

## 📁 Dataset

The dataset contains information about used cars including:

- Car Name
- Year
- Mileage
- Fuel Type
- Transmission
- Engine
- Selling Price

Total records: **8190 rows**

---

## ⚙️ Workflow

1. Data Cleaning
2. Exploratory Data Analysis (EDA)
3. Feature Engineering
4. Model Training
5. Model Evaluation
6. Price Prediction

---

## 🤖 Machine Learning Models Used

- Linear Regression
- Decision Tree Regressor
- Random Forest Regressor

---

## 📈 Results

- Model achieved good accuracy after tuning
- Random Forest performed the best among all models
